# About

A music website that provides a current picture, biography, events and songs of your favorit artists. Also a recomendation for similar artists that you might like as well.

# Technology Stacks

- Last.fm API
- Discogs API
- HTML/CSS
- JavaScript
- jQuery
- Moment.js
- Google Fonts
